# Future Ideas

Practical extensions that fit this starter well:

- supporter signup integrations for Action Network, Mailchimp, NationBuilder, or bespoke APIs
- issue-specific resource library with filters and downloadable packs
- dedicated FAQ page generated from the existing FAQ collection
- press page with statements, media contacts, and downloadable logos
- supporters page with coalition members, endorsements, or case studies
- resources page for toolkits, briefings, and downloadable PDFs
- richer OG image generation driven by content frontmatter rather than one generic SVG
- CSV or CMS import script for quotes, FAQs, or resources
- multilingual content collections for Welsh and English
- accessible campaign timeline component
- petition or pledge flow with serverless form handling
- analytics consent controls if a campaign needs more than Web Analytics
- optional search over blog posts and resources
- printable one-page briefing layout
- MP lookup result caching with KV if scale or rate limits become an issue
- postcode-to-local-council lookup alongside MP lookup for local campaigns
- downloadable social packs generated from campaign theme tokens
